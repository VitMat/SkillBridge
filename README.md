# SkillBridge

### Apprenticeship Matching & Employability Tracking System

**PPOC Policy Conclave 2026 — Hackathon Project**

---

## About the Project

SkillBridge is a simple web prototype that connects **students from public institutions** with **employers offering apprenticeship opportunities** using a **skill-based matching system**.

The project shows how technology can help improve the **education-to-employment transition** in India’s public education system.

---

## Problem

Many students in government colleges, ITIs, and polytechnics face difficulties in finding employment because:

* Career guidance is limited
* Employer connections are weak
* Apprenticeship opportunities are not easily accessible
* Institutions cannot track employment outcomes

SkillBridge demonstrates a possible **digital solution** to this problem.

---

## Features

### Student Portal

* Create student profile
* Enter skills
* Register for opportunities

### Employer Portal

* Post apprenticeship opportunities
* Add required skills

### Matching System

* Matches students with employers based on skills

### Dashboard

* Shows placement results
* Tracks participation

3. Use the navigation menu to explore the system.
